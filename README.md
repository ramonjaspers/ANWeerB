# ANWeerB
ANWB assignment ReactNative weather app

TASKS
- [ ]Geef weerinformatie op dmv https://openweathermap.org/api.
- [ ]Toon de huidige weersomstandigheden voor een specifieke locatie.
- [ ]Toon de temperatuur, het weericoon, de windsnelheid en een korte beschrijving van het weer.
- [ ]Voeg een zoekfunctie toe waarmee gebruikers de weersomstandigheden voor verschillende locaties kunnen opvragen.
- [ ]Zorg ervoor dat de app er goed uitziet op zowel iOS als Android apparaten.
